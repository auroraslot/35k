# LeetCode 60天刷题计划 - 进度追踪器

一个基于HTML、CSS和jQuery的静态页面，用于跟踪LeetCode刷题进度。

## 功能特性

- 📊 **进度统计**：实时显示总题目数、已完成数和完成率
- 📁 **文件上传标记**：通过上传代码文件自动标记题目完成状态
- 💾 **本地存储**：使用浏览器本地存储保存完成状态，刷新页面不丢失
- 🎨 **美观界面**：现代化的UI设计，支持响应式布局
- 🔍 **智能识别**：支持多种文件命名格式识别题目编号

## 使用方法

1. **下载文件**：
   - 下载 `leetcode-tracker.html` 文件
   - 在浏览器中打开该文件

2. **上传代码文件**：
   - 点击"选择文件"按钮，选择你的LeetCode代码文件
   - 支持的文件格式：`.js`, `.py`, `.java`, `.cpp`, `.c`, `.go`, `.rs`, `.ts`, `.php`, `.rb`, `.swift`, `.kt`, `.scala`
   - 支持的文件命名格式：
     - `1.js` (直接题目编号)
     - `problem_1.py` (problem_前缀)
     - `leetcode_001.java` (leetcode_前缀)
     - `001_two-sum.js` (任意数字前缀)

3. **查看进度**：
   - 已完成的题目会高亮显示并标记✓符号
   - 顶部显示总体进度统计和进度条

## 注意事项

- 这是一个纯静态页面，无需服务器
- 所有数据存储在浏览器本地，不会上传到任何服务器
- 建议定期备份你的代码文件到其他位置
