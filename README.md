# LeetCode 60天刷题计划 - 进度追踪器

一个基于HTML、CSS和jQuery的静态页面，用于跟踪LeetCode刷题进度。

## 功能特性

- 📊 **进度统计**：实时显示总题目数、已练习题目数、练习次数和完成率
- 🎯 **直接标记**：点击按钮直接标记题目练习状态，支持多次练习记录
- ➕➖ **计数器控制**：每个题目都有独立的练习次数计数器
- 💾 **本地存储**：使用浏览器本地存储保存练习状态，刷新页面不丢失
- 🎨 **美观界面**：现代化的UI设计，支持响应式布局
- 📤 **数据导出**：支持导出JSON格式的完整练习数据

## 使用方法

1. **下载文件**：
   - 下载 `leetcode-tracker.html` 文件
   - 在浏览器中打开该文件

2. **标记练习进度**：
   - **标记完成**：点击题目行的"标记完成"按钮
   - **增加练习次数**：点击题目行的"+"按钮
   - **减少练习次数**：点击题目行的"-"按钮
   - **完全移除记录**：当练习次数为1时，点击"-"按钮直接移除记录

3. **批量操作**：
   - **全部标记为1次**：将所有未练习的题目标记为练习1次
   - **清空所有记录**：移除所有练习记录
   - **导出数据**：下载JSON格式的进度数据

4. **查看进度**：
   - 已练习的题目会高亮显示并显示练习次数
   - 顶部显示总体进度统计（包含总练习次数）
   - 实时更新的进度条
